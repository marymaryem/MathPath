Nombre Premier
Un entier n > 1 est premier s'il n'a que 2 diviseurs : 1 et lui-même.
Le résultat apparaîtra ici
def premier(x): if x < 2: return False for i in range(2, x // 2 + 1): if x % i == 0: return False return True
〰️
Nombre Ondulant
Un entier est ondulant si ses chiffres alternent : a<b>a<b… ou a>b<a>b…
Le résultat apparaîtra ici
def ondulant(x): ch = str(x) if len(ch) < 3: return False for i in range(len(ch) - 2): if not (ch[i] == ch[i+2] and ch[i] != ch[i+1]): return False return True
💎
Nombre Parfait
Un entier est parfait si la somme de ses diviseurs propres est égale à lui-même. Ex : 6 = 1+2+3.
Le résultat apparaîtra ici
def parfait(n): s = sum(i for i in range(1, n) if n % i == 0) return s == n

Nombre Lisse
Un entier n est lisse si son plus grand facteur premier est ≤ √n.
Le résultat apparaîtra ici
import math def lisse(x): pg = 2 for i in range(1, x): if x % i == 0 and premier(i) and i > pg: pg = i return pg <= math.sqrt(x)
✂️
Nombre Semi-Premier
Un entier est semi-premier s'il est le produit de deux nombres premiers (pas forcément distincts). Ex : 15 = 3×5.
Le résultat apparaîtra ici
def semi_premier(n): for i in range(2, n): if n % i == 0: j = n // i if premier(i) and premier(j): return True return False

Nombre de Mersenne
Un nombre de Mersenne a la forme 2p − 1 où p est premier. Ex : 7 = 2³ − 1.
Le résultat apparaîtra ici
def mersenne(n): k = 2 while 2**k - 1 < n: k += 1 return 2**k - 1 == n and premier(k)

Nombre Super Pair Plus
Un entier est super pair plus si lui-même, tous ses chiffres, et tous ses diviseurs sont pairs.
Le résultat apparaîtra ici
def super_pair_plus(n): # le nombre lui-même doit être pair if n % 2 != 0: return False # tous les chiffres doivent être pairs for d in str(n): if int(d) % 2 != 0: return False # tous les diviseurs doivent être pairs for i in range(1, n): if n % i == 0 and i % 2 != 0: return False return True

PGCD
Plus Grand Commun Diviseur — par différence successive ou division euclidienne.
Le résultat apparaîtra ici
# Méthode par différence def pgcd_diff(x, y): while x != y: if x > y: x -= y else: y -= x return x # Méthode euclidienne def pgcd_euc(x, y): while y != 0: x, y = y, x % y return x
🔀
PPCM
Plus Petit Commun Multiple de deux entiers.
Le résultat apparaîtra ici
def pgcd(x, y): while y != 0: x, y = y, x % y return x def ppcm(x, y): return (x * y) // pgcd(x, y)
❗
Factoriel
n! = 1 × 2 × 3 × … × n
Le résultat apparaîtra ici
def factoriel(n): f = 1 for i in range(1, n + 1): f *= i return f
📈
Puissance
Calcule xy.
Le résultat apparaîtra ici
def puissance(x, y): p = 1 for _ in range(abs(y)): p *= x if y < 0: return 1 / p return p

Combinaison C(n, p)
Nombre de façons de choisir p éléments parmi n. C(n,p) = n! / (p! × (n−p)!)
Le résultat apparaîtra ici
Requis : 0 ≤ p ≤ n
def combinaison(n, p): return (factoriel(n) // (factoriel(p) * factoriel(n - p)))

Conversion de Base
Convertir entre base 10 et une base N (2 à 36).
Le résultat apparaîtra ici
# Base N → décimal def vers_decimal(ch, b): n, p = 0, 1 for c in reversed(ch.upper()): v = int(c) if c.isdigit() else ord(c) - 55 n += v * p p *= b return n # Décimal → base N def vers_base(n, b): ch = "" while n != 0: r = n % b ch = (str(r) if r < 10 else chr(r + 55)) + ch n //= b return ch or "0"
